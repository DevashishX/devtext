# Devtext  - text editor in C language.



### Devtext has the following Features:

* Open, Read, Write text file

* Runs in Terminal

* Accepts Command Line Arguments

* Use of Curser Keys to Navigate 

* Undo and Redo

* Find and Replace

* Syntax Highlighting

* Pageup and Pagedown
